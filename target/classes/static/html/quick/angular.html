Angular快速建站指引

新手教程：https://www.runoob.com/angularjs/

Step1：复制JS框架

Step2：导入

<script src="js/angular.min.js"></script>
<script src="js/angular-route.min.js"></script>

导入两个JS并修改成正确的路径，不使用Angular路由功能可以不导入第二个。

Step3：初始化并开始你的表演

固定初始化写法 JS部分：

<script>
    angular.module('应用名称', ['ngRoute']).controller('控制器名称', function ($scope, $http) {
        //这里开始你的表演...
    });
</script>

固定初始化写法 HTML部分：

<div ng-app="应用名称" ng-controller="控制器名称">
    <!-- Angular作用范围-->
</div>
