Mysql快速建站指引

Step1：maven配置

<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java </artifactId>
    <version>8.0.19 </version>
</dependency<

Step2：配置

以使用SpringBoot为基础，配置在application.yml为例

spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/sakura?serverTimezone=UTC&useUnicode=true&characterEncoding=utf8&useSSL=false #数据库连接，端口号后为数据库名称
    username: root #数据库用户名
    password: 123456 #数据库密码
